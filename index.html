What is this?

CharBrowser is a character viewer for your EQ Emulator server. Once installed your users can look up each other's characters and see their Inventory, Bank, Alternate Abilities, Flags, Keys, Factions and Skills. In addition to viewing the profiles there is a Bazaar viewing utility, a Character Mover (which moves the characters location to one specified by the site admin), a signature builder that generates dynamic images (with character stats) to be used as a forum signature and an API to give developers programatic access to this data. User's can customize their display by interacting with a quest NPC on your server.

Installation

  • Download the files.
  • Copy the contents of the /Project Files/ directory into a path on your web server.
  • Config:
    • If this is a new installation, rename /includes/config.template to config.php.
    • If you are overwriting an existing installation you can preserve your old config.php and delete config.template.
  • Open /includes/config.php in a text editor. In the "SQL Settings" section you will modify the four variables to reflect your own database settings.
  • In a web browser navigate to the path you installed the files to and everything should be running.

Extended Configuration

You can customize the extended features of the tool in the config.php. Some basic knowledge about php variable and array syntax is required.

  • General Settings

    • numToDisplay: how many search results will be displayed.
    • highlightgm: A value of 1 will highlight GM inventories in gold; 0 will not highlight.
    • blockbazaar: A value of 0 enables the bazaar; 1 will disable.

  • Title Settings

    • mytitle: Sets the title of the tool (a lot of people use their server name).
    • subtitle: The subtitle that shows up after the title in a few places.
    • version: This is displayed in a few locations. It's just the build version--you probably don't need to change this.
    • titlefont: The font name you want to use for the title. It must match the name of a file (without extension) in the /fonts/ directory.
    • titlefontR: The Red component of the titles RGB value.
    • titlefontG: The Greencomponent of the titles RGB value.
    • titlefontB: The Bluecomponent of the titles RGB value.
    • titlefontsize: The font size of the title.

  • Character mover settings

    • blockcharmove: A value of 1 will disable to character movier; 0 enables it.
    • charmovezones: This is a two dimensional array of locations your users can move their characters to. The index of the first dimension should be the short zone name (from the database). That points to a three element array with indexes of 'x', 'y' and 'z' which hold the move coordinates for that zone.

  • Permissions: for all permissions a value of 0 displays and a value of 1 hides.

    • PUBLIC: If a user has set the display to public using the quest NPC, their character will display with these settings.
    • PRIVATE: If a user has set the display to private using the quest NPC, their character will display with these settings.
    • GM: If a character is not set to public or private, and they are a GM they will display with these settings.
    • ROLEPLAY: If a character is not set to public or private, arent a GM, and they are Roleplaying they will display with these settings.
    • ANON: If a character is not set to public or private, arent a GM, and they are Anonymouse they will display with these settings.

    • ALL: All characters that haven't met the previous criteria will use these settings.

    • inventory: Display the characters main inventory.

    • coininventory: Display the characters coinage in their inventory.
    • coinbank: Display the characters coinage in their bank.
    • bags: Display the characters bags.
    • bank: Display the characters bank contents.
    • corpses: Display the characters corpses.
    • flags: Display the characters flags.
    • AAs: Display the characters alternate abilities.
    • factions: Display the characters flags.
    • advfactions: Display the characters factions with more advanced numeric values.
    • skills: Display the characters non-language skills.
    • languageskills: Display the characters language skills.
    • keys: Display the characters keys.
    • signatures: Display the characters signatures (which can display some items/stats).

  • Ad Banners

    • adscript: if you use advertisement banners you can embed the HTML for your banner here and it will be added to every page.

  • DATABASE STATS

    • DB_PERFORMANCE: if you uncomment this constant it will cause each page to dump out performance stats on how it interacts with the database. This will be publicly viewable.
